# ABR API Reference

## RPC Commands

### Blockchain
- `getinfo` - Get network information
- `getblockchain` - Get blockchain parameters
- `getblock <hash>` - Get block details
- `getblockhash <height>` - Get block hash at height

### Mining
- `getmininginfo` - Get mining information
- `getnetworkhashps` - Get network hashrate

### Network
- `getnetworkinfo` - Get network info
- `getpeerinfo` - Get connected peers

### ABR-Specific
- `getgenesis` - Get genesis block details
- `getaddress` - Get genesis address
